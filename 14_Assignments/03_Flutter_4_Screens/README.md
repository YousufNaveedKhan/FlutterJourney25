# Assignment 03 – Flutter 4 Screens App

## Task
Build a Flutter app with **4 screens**:

### 1. Signup Screen
- Form with 6 fields: `First Name`, `Last Name`, `Email`, `Contact`, `Password`, `Confirm Password`.
- Signup button shows SnackBar: "Account registered successfully."
- Below form: "Already have an account? Login" (clickable using GestureDetector, navigates to Login).

### 2. Login Screen
- Fields: Email, Password.
- Option: "Forgot Password?"
- Below login button: "Don’t have an account? Signup" (clickable, navigates to Signup).
- On login, navigate to **Chat Screen**.

### 3. Forgot Password Screen
- TextField: "Enter your email".
- Button: Find Account.
- Navigation comes from tapping "Forgot Password?" on Login.

### 4. Chat Screen
- Use `ListView.builder` for chat list.
- Each item: Leading, Title, Subtitle, Trailing.
- On tap: SnackBar "Opening Chat…", then after 3s: "Failed Opening Chat…".
