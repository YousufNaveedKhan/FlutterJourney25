# Assignment 01 – Student Marks Program (Dart)

## Task
Create a Dart program that:
- Stores a student’s **name**, **age**, and **marks in 3 subjects**.
- Calculates **total** and **average** marks.
- Displays the **result** with a **pass/fail message**.

## Requirements
- Use **variables**, **data types**, **operators**.
- Use **print statements** and **concatenation** for output.
- Apply **basic logic** for pass/fail.
