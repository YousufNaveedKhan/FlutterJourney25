# Flutter Environment Setup

Goal of this lecture:
> Set up Flutter correctly and run your first app.

---

## Required Tools

- Flutter SDK
- Android Studio or VS Code
- Android Emulator or Real Device
- Git (recommended)

---

## Step 1: Install Flutter SDK

1. Visit: https://flutter.dev
2. Download Flutter SDK
3. Extract it to a safe location  
   Example:
```

C:\flutter

```

---

## Step 2: Add Flutter to PATH

Add Flutter `bin` folder to system environment variables.

Example:
```

C:\flutter\bin

````

---

## Step 3: Verify Installation

Run:
```bash
flutter doctor
````

This command shows:

* What is installed
* What needs fixing

Fix issues one by one.
No need to panic.

---

## Step 4: Code Editor Setup

### VS Code

* Install Flutter extension
* Install Dart extension

### Android Studio

* Install Flutter plugin
* Install Dart plugin

---

## Important Rule for Students

> If `flutter doctor` is green,
> your Flutter setup is correct.

Small issues are part of learning.
They will be solved during the course.
