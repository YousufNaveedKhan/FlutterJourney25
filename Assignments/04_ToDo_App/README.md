# Assignment 04 – Flutter To-Do App

## Task
Build a Flutter To-Do App to **Add, View, Edit (placeholder), and Delete tasks**.

### App Requirements
1. **Input Fields**:
   - Task Title
   - Task Description
   - Button: Add Task

2. **Add Task**
- On tap, task appears in ListView below.
- Each item: ID (Avatar), Title, Description, Edit & Delete icons.
- SnackBar: "Task Added Successfully!"

3. **Delete Task**
- Remove task on Delete icon tap.
- SnackBar: "Task Deleted Successfully!"

4. **Edit Placeholder**
- On Edit icon tap: SnackBar "Edit Functionality will be added soon!"

5. **Clear All Tasks**
- Button below Add Task: "Clear All Tasks" / "Delete All Tasks"

### UI Requirements
- AppBar: "My To-Do App"
- Use Padding, Icons, Colors, Elevation for a clean, modern look.
